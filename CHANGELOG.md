# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-02

### Added

- Initial release of Cursor Rules Collection
- Project rules for 11 categories:
  - Folder structure (layer-based architecture)
  - TypeScript types and type safety
  - HTTP requests and API layer
  - UI components (Vue 3 Composition API)
  - State management (Pinia)
  - Routing (Vue Router)
  - Forms and validation (Element Plus)
  - Testing patterns
  - Performance optimization
  - GitHub workflow
  - Localization (i18n)
- User rules for global Vue 3 + TypeScript preferences
- Comprehensive README with usage instructions
- Rules summary document
- MIT License

### Project Rules Features

- Layer-based folder structure guidelines
- TypeScript-first approach (no `any` types)
- Service layer for all API calls
- Component composition patterns
- Pinia store best practices
- Performance optimization guidelines
- Collaboration and Git workflow rules

### User Rules Features

- Vue 3 Composition API enforcement
- TypeScript mandatory everywhere
- Code style preferences (arrow functions, early returns, DRY)
- Naming conventions for all file types
- Error handling patterns
- Communication preferences with AI
